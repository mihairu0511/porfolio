実装の仕組み

1. Bug Blaster の操作とダーツの発射

  • Bug Blaster は D-pad を使用して画面下部を左右に移動 できます。
  • A ボタンを押すと上方向にダーツを発射 します。
  • ダーツは Centipede または Mushroom に衝突するか、画面上部に到達 すると消えます。

2. Segmented Centipede の動作

  • Centipede は複数のセグメントで構成 され、各セグメントはスプライトで表現 されています。
  • 画面上部からスタートして左右に移動 します。
  • 画面端に到達するか、Mushroom に衝突すると 1 段階下に降りて反対方向に進行 します。
  • ダーツが当たると、そのセグメントが Mushroom に変化 します。
  • Centipede の頭 (Head) は特別なスプライト で表示され、頭にダーツが当たった場合は 100 点、通常のセグメントは 10 点 のスコアが加算されます。

3. Mushroom Field の配置と動作

  • Mushroom は Centipede の動きを妨害する障害物 として配置されます。
  • ゲーム開始時にランダムな位置に配置され、各レベルごとに増加 します。
  • レベル 1：32 個 / レベル 2：64 個
  • ダーツが 4 回当たると破壊 され、1 点 のスコアが加算されます。

4. スコアの表示と計算

  • 画面上部にスコアを表示 し、プレイ中に常に確認 できます。
  • 次のアクションでスコアが加算 されます。
    • Centipede Head を破壊 → 100 点
    • Centipede Segment を破壊 → 10 点
    • Mushroom を破壊 → 1 点
  • スコアは次のレベルに進んでもリセットされません。

5. ステートマシンと複数レベルの実装

  • START, GAME, PAUSE, WIN, LOSE の 5 つの状態を管理 しています。
  • START 状態：A ボタンを押すと GAME 状態に遷移 します。
  • GAME 状態：Bug Blaster を操作して Centipede を倒す 状態です。
  • PAUSE 状態：START ボタンを押すと一時停止。再度 START ボタンで続きから再開。
  • WIN 状態：Centipede をすべて倒すと WIN 状態に遷移。2 レベル目をクリアでゲームクリア。
  • LOSE 状態：Centipede が画面下部に到達 すると LOSE 状態に遷移。
  • WIN, PAUSE, LOSE 状態 から START 状態に戻る ことができます。
