GBA で音楽を再生してみる。

GBA (Game Boy Advance) の Mode0 を用いて、レトロなドット絵の見た目と操作感を再現しています。プレイヤーはキャラクターを操作して、ムカデを倒すことが目的です。複数の状態 (START, GAME, PAUSE, WIN, LOSE) を管理するステートマシンを実装し、2レベルのステージをクリアして WIN 状態になるか、ムカデが画面下部に到達して LOSE 状態になるまでプレイが続きます。

機能の概要

  • 音楽の再生・停止
    • A ボタン を押すことで音楽を再生、もう一度押すことで停止。
  • 複数の音楽の保存・選択
    • 複数の音楽データを GBA 内に保存し、選択して再生可能。
    • 十字ボタンを用いて、次の曲や前の曲に切り替え。
  • 曲名の表示
    • 曲名を画面に表示し、再生中の曲を確認可能。

実装の仕組み

音楽の読み込みと再生準備

  • 複数の音楽データを GBA のメモリに保存し、選択した曲を再生。
  • A ボタン で再生・停止を切り替え。
  • 曲の切り替えは、十字キーの左右で行い、選択した曲のデータを読み込む。

曲名の表示と切り替え

  • 曲の切り替えは、十字ボタンの左右で行い、次の曲または前の曲に変更。
  • 曲名の表示は、現在の曲インデックスに応じて画面を更新。

画面表示と UI

  • Mode 4を用いて、ピクセルアートのような UI を構成。
  • 曲名をピクセルを用いて表示。
