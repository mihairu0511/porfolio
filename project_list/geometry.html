Figure Rush (ジオメトリーダッシュをGBAに移植）

ジオメトリーダッシュというゲームをご存知でしょうか。海外で人気な音楽に合わせて障害物を避けながら進むリズムアクションゲームです。今回は C言語と VSCode/Docker を使用して、このゲームのクローン (Figure Rushと命名) を Game Boy Advance (GBA) プラットフォーム向けに再現してみました。ゲーム開発ツールを一切使用せず、ハードウェア固有の技術を活用して、レトロゲームの雰囲気を忠実に再現しています。さらに、実機の GBA にも移植し、実際にプレイ可能な状態にしています。

プロジェクトの特徴

  • 開発環境と言語
    • C言語 を使用した低レベルなハードウェア操作。
    • VSCode と Docker を利用したクロスコンパイル環境。
  • 実機 GBA への移植と動作確認
    • フラッシュカートリッジを用いて、GBA の実機に ROM データを転送しました。
    • エミュレーターだけでなく、実際の GBA 本体での動作確認を行い、ハードウェア互換性の検証を実施しました。
  • GBA ハードウェアの特性を活かした最適化
    • GBA のハードウェアリソースを最大限に活用するため、以下の技術を駆使しています:
      • モード 0 を使用したスプライト描画と背景レイヤーの制御。
      • ポインタ操作などでの VRAM の直接操作による、高速なグラフィック更新。
  • ゲームデザインとメカニクス
    • タイルとスプライトのデザイン
      • Geometry Dash のようなシンプルなデザインを再現しました。
      • ドット単位で自作したタイルとスプライトを使用しています。
    • スプライトアニメーションと衝突判定
      • ピクセル単位の衝突判定を実装し、プレイヤーの操作感を強化しました。
      • 複数のアニメーションステートを管理することで、リアルな動きを表現しています。
    • 重力と物理シミュレーション
      • リアルな重力の動きをシミュレートし、プレイヤーキャラクターのジャンプと落下に違和感がないよう設計しました。
      • 速度と加速度の計算 を行い、滑らかな物理挙動を実現しています。
    • ゲーム内背景の設計
      • XL 背景を用いて、横スクロールのダイナミックな背景を表現しました。
      • 背景のパララックス効果を取り入れ、奥行き感のあるゲーム体験を提供しています。
