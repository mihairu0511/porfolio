概要：

このボコーダーは、Max/MSP を使用して作成されており、FFT（高速フーリエ変換） を用いて音声信号を周波数帯域に分解し、キャリア信号 と モジュレーター信号 を組み合わせることで、ロボットボイスやシンセサイザーボイスを生成します。

機能の概要：

  • オーディオ入力
    • マイクを使用してリアルタイムの音声入力
    • オーディオファイルを読み込んで再生
  • FFTとIFFTによる周波数分析・合成
    • pfft~ オブジェクトを使用して、FFT と IFFT を実行
    • 各周波数帯域ごとの 振幅 と 位相 を取得
  • リングモジュレーション / 周波数変調 / 振幅変調
    • RM（Ring Modulation）：入力信号とキャリア信号を掛け合わせ、金属的な音を生成
    • FM（Frequency Modulation）：キャリア信号の周波数を変調し、複雑な倍音を生成
    • AM（Amplitude Modulation）：キャリア信号の音量を変調し、音の揺らぎを作成
  • キャリア信号の選択
    • ノコギリ波 (Sawtooth)
    • ノイズ (Noise)
    • 外部音源（マイクまたはオーディオファイル）
  • シェーダーリバーブ
    • p Shroder を使用して、シェーダーリバーブを追加
    • 音に奥行きを与え、空間的な広がりを作成
  • マスターゲインと出力
    • 最終的な音量を調整する Master Gain
    • スペクトラムアナライザーでリアルタイムに周波数スペクトルを視覚化

実装の仕組み：

  • FFT と IFFT の使用
    • pfft~ オブジェクトを使用して、FFT（高速フーリエ変換）を実行
    • 各帯域ごとに 振幅 と 位相 を取得
    • IFFT（逆フーリエ変換）を用いて、処理後の信号を再構成
  • 周波数帯域ごとのエンベロープの適用
    • 各周波数帯域の振幅エンベロープを取得
    • モジュレーター信号のエンベロープをキャリア信号に適用してボコーダー効果を実現
  • キャリア信号の生成と切り替え
    • ノコギリ波 (Sawtooth) や ノイズ (Noise) をキャリア信号として生成
    • MIDI キーボードからの入力を受け取り、mtof オブジェクトで周波数に変換
    • selector~ オブジェクトを使用して、キャリア信号を動的に切り替え
  • リングモジュレーション / 周波数変調 / 振幅変調
    • リングモジュレーション (RM)：キャリア信号とモジュレーター信号を掛け合わせ、倍音を強調
    • 周波数変調 (FM)：モジュレーター信号の振幅をキャリア信号の周波数に適用し、倍音の変化を作成
    • 振幅変調 (AM)：モジュレーター信号のエンベロープをキャリア信号の音量に適用
  • シェーダーリバーブの追加
    • p Shroder を使用して、シェーダーリバーブを適用し、音の残響効果を追加
    • ライブゲイン (live.gain~) でリバーブ量を調整
  • GUI と操作方法
    • ControlP5 を用いて、直感的な GUI（ボタン、スライダー）を実装
    • キャリア信号の切り替え、フィルターのオン/オフ、モジュレーションの深さの調整をリアルタイムで操作可能
