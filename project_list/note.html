モジュラーオーディオシンセサイザー

この Python スクリプトは、モジュラーオーディオシンセサイザーです。 NumPy と SciPy ライブラリを使用して、波形生成、FM 合成、コード進行を用いた音楽の生成と再生を行います。 また、ADSR エンベロープ、周波数変調 (FM)、ディレイエフェクト、コード合成 などの高度な機能も搭載しています。

1. 音符と音長の変換 (Note and Duration Conversion)

機能の概要:

  • 音符を周波数に変換 (Note to Frequency Conversion)
    • 音符 (C4, D#4, A5, など) を MIDI ノート番号を用いて周波数 (Hz) に変換します。
    • 基準音 A4 を 440 Hz として、12 音階の対数関数を使用して周波数を計算。
    • rest の場合は 0 Hz に設定し、無音を表現します。
  • 音符の長さをミリ秒に変換 (Duration to Milliseconds)
    • BPM (Beats Per Minute) を基準に、音符の長さ (1/4, 1/8, 1/2 など) をミリ秒 (ms) に変換します。
    • デフォルトでは 4分音符 を1拍として計算。

2. ADSR エンベロープ (ADSR Envelope)

機能の概要:

  • ADSR (Attack, Decay, Sustain, Release) を用いたエンベロープ処理により、波形の音量変化を滑らかにします。
    • A (Attack)：音の立ち上がり時間
    • D (Decay)：ピークからサステインレベルまで減衰する時間
    • S (Sustain)：保持する音量の割合
    • R (Release)：音を止めた後の減衰時間
  • エンベロープを掛けることで、クリックノイズの削減や、自然な音の減衰を実現。

技術的なポイント:

  • エンベロープは、音の立ち上がり (Attack) から音の減衰 (Release) まで、音のボリュームを滑らかに調整するための関数です。
  • NumPy を使用して、linspace() 関数でリニアに音量を変化させています。

3. 波形生成 (Waveform Generation)

機能の概要:

  • 以下の 4 種類の波形 を生成することができます:
    • サイン波 (Sine Wave): 最も基本的な滑らかな波形
    • 鋸歯状波 (Sawtooth Wave): 奇数・偶数倍音を含む、鋭い音色
    • 矩形波 (Square Wave): 奇数倍音のみを含み、パルス的な音色
    • 三角波 (Triangle Wave): 奇数倍音を含むが、倍音成分が少ない柔らかい音色

技術的なポイント:

  • NumPy の sin() 関数を用いてサイン波を生成。
  • SciPy の sawtooth() や square() 関数を用いて、鋸歯状波、矩形波、三角波を生成。
  • サンプルレート (Sample Rate) を 44100 Hz に設定して、高品質な音声を生成。

4. コード生成 (Chord Generation)

機能の概要:

  • コード進行 (Chord Progression) を生成します。
  • 複数の音符を重ねて和音 (Chord) を生成 します。
  • 和音の例: C メジャーコード → ['C4', 'E4', 'G4']

技術的なポイント:

  • 加算合成 (Additive Synthesis) を使用して、複数の音符の波形を重ね合わせています。
  • 正規化 (Normalization) を行い、音量のクリッピング (歪み) を防いでいます。

5. FM 合成 (Frequency Modulation Synthesis)

機能の概要:

  • 周波数変調 (FM) により、複雑な倍音を含む音色を生成。
  • キャリア波 (Carrier Wave) の周波数を モジュレーター波 (Modulator Wave) で変調します。
  • FM シンセサイザー として、金属的でダイナミックな音色を再現。

技術的なポイント:

  • Modulation Index (I) を設定して、モジュレーションの深さを調整。
  • モジュレーター波にサイン波、矩形波、鋸歯状波、三角波を設定可能。
  • キャリア周波数 と モジュレーター周波数 の比率によって、倍音成分が変化。

6. ディレイエフェクト (Delay Effect)

機能の概要:

  • ディレイ (Delay) を用いて、エコー効果 を実現。
  • オフセット時間 (offset)、リピート回数 (repeats)、減衰率 (A) を指定可能。

技術的なポイント:

  • ディレイ信号を ゼロパディング (Zero Padding) によって遅延させています。
  • 減衰率 (A) を設定することで、繰り返しごとに音量が小さくなるエコーを再現。

7. メロディーとコードの再生 (Melody and Chord Playback)

機能の概要:

  • メロディー と コード進行 をそれぞれ生成し、合成して再生します。
  • 2 つのメロディーライン と 1 つのコード進行 を同時に演奏可能。
  • メロディー、メロディー 2、コード の 3 つのレイヤーを重ね合わせて楽曲を構成。

技術的なポイント:

  • 各レイヤー (メロディー、メロディー 2、コード) は、音符の配列 (list of tuples) として指定。
  • 各音符に対して、音程 (note) と 音長 (duration) を設定可能。
  • 生成した波形を NumPy の concatenate() 関数を用いて連結しています。

8. デモンストレーション (Demo)

デモ 1: シンプルな 2 重奏

  • メロディー 1 と メロディー 2 の 2 パートから構成。
  • サイン波 (Sine Wave) を用いた柔らかい音色。
  • BPM 100 に設定して、ゆったりとしたテンポ。

デモ 2: FM 合成による鋭い音色

  • FM 合成 を使用して、金属的で複雑な倍音を含む音色を生成。
  • キャリア波 を サイン波 (Sine Wave) に設定。
  • モジュレーター周波数 を 15 Hz に設定し、うねりのある音色を表現。
  • ディレイエフェクト を追加し、エコーのある空間的な効果を再現。

デモ 3: クラシック風の和音進行

  • クラシック音楽風のコード進行 と メロディー を組み合わせています。
  • FM 合成 を用いた複雑な倍音構成。
  • 三角波 (Triangle Wave) をベースにした柔らかい音色。
  • ディレイエフェクト を用いて、幻想的な響きを演出。

9. 機能の応用と可能性 (Applications and Potential)

音楽制作:

  • このシンセサイザーは、コード進行とメロディーを自由に設計できるため、音楽制作に応用可能。
  • FM 合成やディレイエフェクトを組み合わせて、複雑でダイナミックな音色を作成できます。

教育ツール:

  • 倍音 (Harmonics) の構成や波形の種類が音色に与える影響を直感的に理解できます。
  • FM 合成 の学習や、ADSR エンベロープの効果を視覚的に確認するための教材としても利用可能。

サウンドデザイン:

  • ゲーム音楽 や 効果音 の作成に応用できます。
  • シンプルな正弦波から複雑な FM 合成まで、幅広い音色の設計が可能。

10. まとめ (Summary)

  • モジュラーオーディオシンセサイザーとして、波形生成、FM 合成、コード進行を用いた楽曲生成を実現。
  • ADSR エンベロープ により、滑らかな音の立ち上がりと減衰を表現。
  • FM 合成 を用いたダイナミックで金属的な音色の生成が可能。
  • ディレイエフェクト を用いたエコー効果を再現し、空間的な広がりを表現。
  • 教育ツール や 音楽制作、サウンドデザイン など、様々な用途に応用可能。
